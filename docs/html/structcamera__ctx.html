Detailed Description

Aggregates all state required for a V4L2 camera streaming session.

This context structure stores file descriptors, V4L2 configuration, buffer metadata, and pointers to memory-mapped frame buffers. All camera operations take a pointer to this context instead of relying on global variables, improving modularity and maintainability.
