Detailed Description

Kernel module providing LED control for camera streaming.

This module exposes simple IOCTL commands that allow a user-space application to control two GPIO-driven LEDs. GREEN for "streaming ON" and RED for "streaming OFF".

The driver registers a character device and responds to:

  • CAM_IOC_START - turn on GREEN LED (stream active)
  • CAM_IOC_STOP - turn on RED LED (stream stopped)
  • CAM_IOC_RESET - reset both LEDs to OFF
