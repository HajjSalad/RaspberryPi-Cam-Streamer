Consume JPEG frames from the pipeline and stream them to the client.

Function Prototypes

Performs the following pipeline stages:

  1. Retrieve a JPEG frame from the shared circular buffer
  2. Send the JPEG frame to the connected client as MJPEG frame
  3. Release associated memory after transmission
Note
This function represents the consumer stage of the producer-consumer streaming pipeline.
Parameters
cctxPointer to the camera context structure that holds all session state
sctxPointer to the stream structure context that holds stream sessions
pipePointer to the pipeline context containing thread and synchronization primitives
Returns
0 on success, negative value on error
