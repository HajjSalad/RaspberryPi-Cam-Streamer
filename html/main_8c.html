Detailed Description

Entry point for the Raspberry Pi MJPEG streaming.

Responsibilities:

  1. Initialize the camera
  2. Start the HTTP server
  3. Setup the multithreaded producer-consumer pipeline

This file coordinates the end-to-end streaming process from camera capture to HTTP MJPEG delivery.
