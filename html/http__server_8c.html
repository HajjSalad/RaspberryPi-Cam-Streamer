Detailed Description

HTTP server handler module.

Implements basic HTTP request handling and response generation. Provides initialization, routing, and utilities for sending responses.

int accept_client_connection ( struct stream_ctx *  sctx)

Accepts an incoming HTTP client connection.

This function blocks until a client connects to the listening socket. On success, the newly accepted client is stored in 'sctx->client_fd'. It also prints the remote client's IP and port.

Parameters
sctxPointer to stream context containing listening server socket.
Returns
The client file descriptor on success, -1 on failure.

Initializes and starts a simple HTTP server for MJPEG streaming.

This function creates a TCP socket, enables address reuse, binds it to the port, and begins listening for incoming client connections. The resulting listening socket is stored in the stream context.
